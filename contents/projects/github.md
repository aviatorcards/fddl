### fddl - Fast Document Deployment Layer

A static site generator written in Swift, featuring a custom markdown processor (SwiftMark), flexible shortcode system, and plugin architecture.

**Technologies**: Swift, SwiftNIO, Markdown  
**Status**: Active Development
[Learn More](/projects/fddl.html)

### SwiftMark

High-performance markdown processor built with a recursive descent parser. Designed for speed and extensibility, SwiftMark powers the fddl static site generator.

**Technologies**: Swift  
**Status**: Integrated within fddl  
[Learn More](/projects/swiftmark.html)
[View Project](https://github.com/aviatorcards/swiftmark)
