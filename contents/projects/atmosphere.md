# Atmosphere

Atmosphere is a lightweight, high-performance journaling app designed for speed and extensibility, built entirely in Swift.

## SwiftMark Integration

At its core, Atmosphere leverages the **[SwiftMark Processor](/projects/swiftmark.html)**, a custom-built Markdown engine that provides real-time, high-fidelity rendering with minimal overhead.

### Key Features

- **Real-time Rendering**: Instant feedback as you type using the SwiftMark engine.
- **Custom Extensions**: Easily extend the processor to support custom syntax and components.
- **Native Performance**: Built with SwiftUI for a fluid, responsive editing experience.

## Tech Stack

- **Language**: Swift
- **UI Framework**: SwiftUI
- **Markdown Engine**: SwiftMark Processor
